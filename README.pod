This is Perl module B<HTTP::UADetector>.


=head1 INSTALLATION

HTTP::UADetector installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan HTTP::UADetector

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install


=head1 DOCUMENTATION

HTTP::UADetector documentation is available as in POD. So you can do:

    % perldoc HTTP::UADetector
